# Cursor.js
**_Cursor.js is lightweight javascript library for text-writing/type-writing effect_**.


# Installation
- Download .zip file and extract cursor.js to your javascript folder and add

```html
        <head>
            <script src="/path/to/cursor.js"></script>
        </head>
```
